## Shared Responsibility Model
For the Linux (ubuntu 24.04) Virtual Machine deployed on Azure, Microsoft is responsible for securing the underlying cloud infrastructure, including physical datacenters, networking, hardware, and the virtualization platform. I am responsible for securing the Ubuntu operating system, managing user access, configuring permissions, installing updates, and protecting any data stored on the virtual machine.
